# letterpad

## summary

教室のレターパッドのTeX styleファイル.
(ワードファイルは事務に言えばもらえるが,
 TeXのスタイルファイルはなかったので,
 自分で作った.)

`letterpad-stylefile/`
にあるファイルをすべて同じフォルダに置いて,
`sample.tex`
をコンパイルすれば
よいと思う.

## 技術的なこと

事務からもらったワードファイルをpdfに変換したもの (`letterpad.pdf`) を,
背景に埋め込んでいるだけ.
ヘッダーの分だけ書き出しを下げる必要があるので,
それを調整している.
様式が変わったならpdfファイルを差し替えるだけでいいと思う.

`\begin{letterpage}`と`\end{letterpage}`の間に,
1ページ分の本文を書く必要がある.
複数ページに渡る場合は,
適当なところで, 環境を切って書き始める.
(自動での改ページはしてくれない)
